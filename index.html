What this is

A RAPP neighborhood is a planted GitHub repo whose collaborators are its members. This neighborhood is for the Microsoft SE Team — outcome-driven automation, distributed across the team. The public gate (this page) carries the identity and the join path. The actual workflow — shared agents, member roster, decision log, twin chat — lives in a private companion repo, gated by GitHub collaborator membership.

Door 1 — Public
Anyone can see the identity, the card, and how to request access.
Door 2 — Members
The private companion at microsoft-se-team-neighborhood-private carries the workflow. If you are a GitHub collaborator there, your brainstem will mount the neighborhood agents on subscribe.

Request access

A current member must add you as a collaborator on the private companion.

If you have a brainstem

Subscribe with one command:

brainstem join https://github.com/kody-w/microsoft-se-team-neighborhood

Your brainstem will read this gate, find the private companion, and check whether your GitHub token has access. If yes — the neighborhood agents mount. If no — it offers to open a join Issue for you.
